# odin-recipes

This will be my summary of lessons learnt during the HTML Recipe Project.

The project required to build a rudimentary recipe website with basic structure.

Some key HTML elements were used to achieve that, such as paragraphs, headings, lists (both orderedvand unordered), links and images.

Knowledge of file structure, absolute and relative paths was reinforced. It was a good opportunity to practice writing proper HTML boilerplate.
